jScrollPane - changelog

jScrollPane was originally developed in December 2006. Since then it has been constantly updated to fix bugs and add new features. The change history since November 2008 is available in the jScrollPane git repository. In August 2010 a major rewrite was undertaken, starting from a blank canvas and adding long awaited features like horizontal scrolling and automatic reinitialisation. If you want to find old versions of jScrollPane then there are links to the still available versions below:

  • 2011-04-17 - 2.0.0beta10 - cleaner required size calculation, improved keyboard support, stickToBottom/Left, other small fixes
  • 2011-01-31 - 2.0.0beta9 - new API methods, bug fixes and correct keyboard support for FF/OSX
  • 2011-01-29 - 2.0.0beta8 - touchscreen support, improved keyboard support
  • 2011-01-23 - 2.0.0beta7 - scroll speed consistent (thanks Aivo Paas)
  • 2010-12-07 - 2.0.0beta6 - scrollToElement horizontal support
  • 2010-10-18 - 2.0.0beta5 - jQuery 1.4.3 support, various bug fixes
  • 2010-09-13 - 2.0.0beta4 - clickOnTrack support, bug fixes
  • 2010-08-27 - 2.0.0beta3 - Horizontal mousewheel, mwheelIntent, keyboard support, bug fixes
  • 2010-08-21 - 2.0.0beta2 - Bug fixes
  • 2010-08-17 - 2.0.0beta1 - Rewrite to follow modern best practices and enable horizontal scrolling, initially hidden elements and dynamically sized elements.
